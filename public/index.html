نظام تسجيل الطلبة (SRS)

أنشأت هذه النسخة لتعمل كخلفية API مبسطة مع SQLite. يمكنك بدء التطبيق ثم استخدام أدوات مثل Postman أو curl.

النقاط الأساسية

  • Users: تسجيل دخول وتسجيل مستخدم جديد.
  • Students: CRUD للطلاب.
  • Courses: CRUD للمقررات.
  • Instructors: إضافة دكاترة وربطهم بالمقررات.
  • Enrollments: تسجيل الطلاب في المقررات.
  • Reports: ملخصات وأرقام.

الواجهة البرمجية (API)

/api/users
/api/students
/api/courses
/api/instructors
/api/enrollments
/api/reports

تشغيل

npm install
